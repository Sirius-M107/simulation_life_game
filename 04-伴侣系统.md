# 伴侣同行者系统

> 本文档定义游戏的伴侣同行者系统：三阶段成长、角色分组、满意度与黑化机制。
>
> 相关文档：[00-总览.md §1.1](./00-总览.md) 核心特色 | [03-事件与驿站.md §7.4](./03-事件与驿站.md) 驿站同伴交谈

---

## 1. 三阶段设计

> 同行者系统采用**三阶段递进**模式：相遇 → 结伴 → 赐福。玩家通过持续投入资源推进关系，最终获得永久赐福效果。

| 阶段 | 名称 | 目标 | 消耗资源 | 阶段转换条件 |
|------|------|------|----------|--------------|
| 阶段 1 | 相遇 | 约会、送礼物 | 金币 + 魔力（MP） | 满意度 ≥ 90 时进入阶段 2 |
| 阶段 2 | 结伴 | 完成仪式 | 金币 + 体力（HP）+ 魔力（MP） | 支付费用后进入阶段 3 |
| 阶段 3 | 赐福 | 维持关系 | 混合资源 | 持续获得赐福效果 |

### 1.1 阶段 1：相遇

- **目标**：通过约会、送礼物提升满意度
- **消耗**：金币 + MP
- **行为选项**：
  - 驿站"与同伴交谈"（消耗少量 MP，恢复少量成就感和影响力）
  - 约会（消耗金币 + MP）
  - 送礼物（消耗金币）
  - 正确选择事件选项（额外提升）
- **转换条件**：满意度 ≥ 90

### 1.2 阶段 2：结伴

- **目标**：完成关系结伴仪式
- **消耗**：金币 + HP + MP（一次性投入较大）
- **转换条件**：支付费用后立即进入阶段 3

### 1.3 阶段 3：赐福

- **目标**：维持关系，享受持续赐福
- **消耗**：混合资源（小额持续投入）
- **效果**：永久赐福效果（见 §2 角色分组）

## 2. 同行者分组

### 2.1 赐福设计原则

> **设计裁决（已讨论）**：赐福效果一律为**机制类**（不采用数值加成 / 减伤叠层），通过改变战斗规则、资源循环或决策选项来体现。

**赐福机制类型**（4 类）：
1. **抽牌 / 资源 / 跳过**：每回合多抽 1 张 / 战斗开始抽 N 张 / 跳过一半费用
2. **减伤 / 抵消 / 反弹**：每次受伤恢复 N 点 / 首次伤害免伤 / 反弹部分伤害
3. **状态 / debuff / 控制**：附加流血 / 破甲 / 怪物每回合下降技能 / 概率晕眩敌人
4. **增益 / 升级 / 重抽**：每场战斗可重抽 1 张 / 变招、追击额外 +1 / 升级标签二选一

**调性原则**：
- **正常组（伊莎贝拉 / 艾琳娜 / 莉莉）** → 保护 / 支援类赐福（类型 1、2、4 为主）
- **黑化组（塔莉娅 / 戴安娜）** → 攻击 / 高风险类赐福（类型 3、4 为主，带副作用）

### 2.2 正常组（保护 / 支援调性）

> 正常组同伴通过**正向投入**提升满意度，满意度越高，赐福触发越频繁或效果越强。

| 名称 | 性格 | 家庭条件 | 初始满意度 | 黑化阈值 | 赐福类型 | 赐福效果 |
|------|------|----------|------------|----------|----------|----------|
| 伊莎贝拉公主 | 温柔体贴 | 良好 | 高 | 低 | 减伤 / 抵消类 | **待设计**：建议方向——首次致命伤害保留 1 HP 或每场战斗开始获得护盾 |
| 艾琳娜领主女儿 | 独立自信 | 中等 | 中 | 中 | 增益 / 重抽类 | **待设计**：建议方向——每场战斗可重抽 1 张或每回合首次攻击 +1 行动点 |
| 神秘女术士莉莉 | 活泼开朗 | 普通 | 中 | 中 | 抽牌 / 资源类 | **待设计**：建议方向——每场战斗开始额外抽 1 张或战利品金币 +N |

### 2.3 黑化组（攻击 / 高风险调性）

> 黑化组同伴**初始满意度低**、**黑化阈值高**（更容易触发黑化），赐福更具攻击性但附带自噬代价。

| 名称 | 性格 | 家庭条件 | 初始满意度 | 黑化阈值 | 赐福类型 | 赐福效果 |
|------|------|----------|------------|----------|----------|----------|
| 黑暗精灵塔莉娅 | 神秘莫测 | 不明 | 低 | 高 | 状态 / debuff 类 | **待设计**：建议方向——每次攻击附加流血 debuff 或怪物每回合降技能 |
| 堕落公主戴安娜 | 难以捉摸 | 未知 | 低 | 高 | 增益 / 高风险类 | **待设计**：建议方向——战斗开始 -10% maxHp，但首次攻击伤害 ×2 |

> **黑化警告**：黑化组如果长期不投入资源，会自动触发黑化剧情，赐福效果转为"暗影"系列（机制更激进但副作用更重，详见 [§4.3](#43-黑化赐福暗影系列)）。

## 3. 满意度系统

### 3.1 满意度定义

- **满意度**：隐藏数值，范围 0-100
- **初始值**：根据性格和家庭条件（60-80，正常组）
- **满意度影响**：满意度越低，黑化率越高

### 3.2 黑化阈值

- 每个同行者有不同的黑化阈值，低于该阈值时触发黑化
- 正常组阈值较低（30-40），黑化组阈值较高（50-55）
- 黑化触发后，赐福效果会**反转**（从增益变为减益，或转向"暗影"系列）

### 3.3 提高满意度的方式

- **驿站与同伴交谈**（消耗少量 MP）
- **约会**（消耗金币 + MP）
- **送礼物**（消耗金币）
- **正确选择事件选项**（在随机事件中做出同伴认可的选择）

### 3.4 降低满意度的场景

- 长时间不投入资源
- 战斗失败 / 资源耗尽
- 在事件中做出同伴不认可的选择

## 4. 赐福效果与战斗加成

> 赐福效果在阶段 3 激活后**永久生效**，影响战斗数值（详见 [02-怪物与战斗.md §3](./02-怪物与战斗.md) 战斗公式）。

### 4.1 赐福效果分类

> **设计裁决（已讨论）**：赐福效果采用**机制类**（不采用数值加成）。本节分类按 [§2.1 赐福设计原则](#21-赐福设计原则) 中的 4 大机制类型组织。

- **抽牌 / 资源 / 跳过类**（莉莉）：每回合多抽 N 张 / 战利品金币 +N / 跳过部分费用
- **减伤 / 抵消 / 反弹类**（伊莎贝拉）：首次致命伤害保留 1 HP / 反弹部分伤害 / 每回合恢复 HP
- **状态 / debuff / 控制类**（塔莉娅）：附加流血 / 破甲 / 怪物攻击力下降 / 概率晕眩
- **增益 / 升级 / 重抽类**（艾琳娜、戴安娜）：每场战斗重抽 1 张 / 首次攻击伤害 ×2 / 变招追击额外 +1

> 赐福效果在阶段 3 激活后**永久生效**，通过改变战斗规则、资源循环或决策选项体现（详见 [02-怪物与战斗.md §3](./02-怪物与战斗.md) 战斗管线）。

### 4.2 赐福叠加

> **设计裁决（已讨论）**：赐福效果采用**机制类**设计，不采用数值加成，因此不再适用"取总和 / 取最高"的数值叠加规则。

- 同一玩家可同时拥有**多位同行者**（每个同行者独立跟踪）
- 机制类效果**可同时生效**（不互斥），如"重抽 1 张"与"额外抽 1 张"叠加为每场战斗重抽 + 多抽
- 同类型机制**不叠加数值**——例如两位同伴若都给"+10% 伤害"是不允许的；改为机制差异（如"首次致命伤害保留 1 HP" + "每回合恢复 5 HP"），机制同时生效
- 同伴的赐福效果**全程永久**，不随战斗切换而消失

### 4.3 黑化赐福（暗影系列）

> 黑化组的赐福转为**暗影机制变体**——更激进的攻击机制 + 更重的自噬代价。

- 黑化组触发黑化后，赐福机制转为"暗影"系列
- 暗影赐福机制**更强**（如"流血 debuff"叠加"中毒" + "怪物攻击力下降"）
- 副作用（自噬代价）：
  - 战斗开始时损失 X% maxHp（X 在 5-15% 之间）
  - 或每回合结束时损失少量 HP / MP
- 可通过"拯救支线"将暗影赐福转回正常赐福（但需额外资源投入）
